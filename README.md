# Course Webpage — Spring 2026

Course webpage for Pang Wei Koh's class at the University of Washington, Spring 2026.

Built with [Just the Docs](https://just-the-docs.github.io/just-the-docs/) and [Just the Class](https://github.com/kevinlin1/just-the-class).

## Local Development

1. Install [Ruby](https://www.ruby-lang.org/en/documentation/installation/) and [Bundler](https://bundler.io/).
2. Run `bundle install` to install dependencies.
3. Run `bundle exec jekyll serve` to build and preview at `localhost:4000`.

## Deployment

The site is configured to deploy via GitHub Pages using GitHub Actions. Push to the `main` branch to trigger a build and deploy.

Go to **Settings > Pages > Build and deployment > Source** and select **GitHub Actions**.
